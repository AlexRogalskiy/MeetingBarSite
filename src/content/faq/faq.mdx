<h2>I have a warnings from Apple and can't install/open</h2>

Apple isn't very friendly to software distributed outside of AppStore.
This app is fully opensource and doesn't have any security issue.
For installing it, you need to:

* Allow installing software outside of AppStore before installation
* Allow opening the app from Preferences after installation

Follow this official from Apple Support to do this:

* <OutboundLink target="_blank" rel="nofollow noopener noreferrer" href={`https://support.apple.com/en-us/HT202491`}> <Underlining color="secondary" hoverColor="secondary"> Safely open apps on your Mac </Underlining> </OutboundLink> from Apple Support Guides

<h2>The application takes a long time to update calendar changes</h2>

The application is updated immediately after changes to the macOS calendar app.
By default, the calendar app update subscribed calendars only every 15 minutes.
But you can reduce it manually:

* <OutboundLink target="_blank" rel="nofollow noopener noreferrer" href={`https://support.apple.com/en-gb/guide/calendar/icl1024/mac`}> <Underlining color="secondary" hoverColor="secondary"> Refresh calendars on Mac </Underlining> </OutboundLink> from Apple Support Guides

<h2>Will there be an app for Windows?</h2>

No. The application based on the macOS Calendar API, which is not compatible with the Windows calendar.
